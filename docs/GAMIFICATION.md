# 게이미피케이션 기획 — Phase 1: 포인트 & 아이템샵

> 작성일: 2026-04-01
> 상태: 기획 확정 (구현 대기)
> 목적: 운동 동기 부여를 위한 보상 시스템
> 범위: **포인트 획득 → 아이템샵 구매 → 쿠폰 사용** (이 흐름만 우선 구현)

---

## 설계 원칙

1. **단순함 우선** — 포인트 벌고, 아이템 사고, 쿠폰 쓰는 3단계만
2. **부부 맞춤** — 보상이 서로에게 향하는 구조 (안마 쿠폰, 설거지 면제 등)
3. **자율 운영** — 아이템과 가격을 부부가 직접 등록/수정

---

## 1. 포인트 시스템

### 1.1 포인트 단위

- 단위: **P** (예: 100P, 500P)

### 1.2 획득 규칙

| 조건 | 포인트 | 설명 |
|------|--------|------|
| 운동 세션 완료 | **100P** | 1일 1회만 인정 (중복 불가) |
| 3일 연속 운동 | **+50P** 보너스 | 3일차 완료 시 자동 지급 |
| 7일 연속 운동 | **+150P** 보너스 | 7일차 완료 시 자동 지급 |
| 30일 연속 운동 | **+500P** 보너스 | 30일차 완료 시 자동 지급 |
| 커플 동시 운동 | **+50P** 각각 | 같은 날 둘 다 운동 완료 시 |

### 1.3 스트릭 보너스 규칙

- 스트릭 보너스는 **해당 일수 도달 시 1회만** 지급
- 예: 7일 연속 달성 → 100P(세션) + 150P(7일 보너스) = 250P
- 3일/7일/30일 보너스는 **중첩 가능** (30일 달성 시 3일+7일+30일 모두 받음)
- 스트릭이 끊기면 1일차부터 다시 시작

### 1.4 일일 최대 획득량

- 하루 최대: **300P** (인플레이션 방지)
- 계산: 세션 100P + 스트릭 보너스(최대 150P) + 커플 보너스 50P = 300P
- 30일 보너스(500P)는 일일 상한 예외 적용

## 2. 아이템샵

### 2.1 기본 제공 아이템 (초기 세팅)

앱 설치 시 아래 아이템이 기본 등록됨. 부부가 가격/내용을 자유롭게 수정 가능.

| 아이템 | 기본 가격 | 설명 |
|--------|-----------|------|
| 카페 쿠폰 | 800P | 카페 음료 1잔 |
| 디저트 쿠폰 | 1,000P | 디저트/베이커리 1회 |
| 치킨 쿠폰 | 1,500P | 치킨 1회 주문 |
| 외식 쿠폰 | 2,000P | 외식 1회 |
| 편의점 쿠폰 | 500P | 편의점 간식 1회 |

### 2.2 커스텀 아이템

- 부부가 직접 **아이템 추가/수정/삭제** 가능
- 입력 항목: 아이템 이름, 가격(P), 설명(선택)
- 가격 범위: 100P ~ 10,000P

### 2.3 구매 & 사용 흐름

```
[아이템샵] → 아이템 선택 → 포인트 차감 → [내 인벤토리]에 저장
                                              ↓
[내 인벤토리] → 쿠폰 선택 → "사용하기" → 파트너에게 알림 → 사용 완료 처리
```

- **구매**: 포인트가 충분하면 즉시 차감, 인벤토리에 추가
- **사용**: 쿠폰 사용 시 파트너에게 푸시/인앱 알림
- **만료**: 쿠폰 만료 없음 (단순하게 유지)
- **환불**: 미사용 쿠폰은 환불 가능 (100% 포인트 반환)

## 4. UI 화면 구성

### 4.1 포인트 표시

- **홈 화면 상단**: 현재 보유 포인트 (예: `1,250P`)
- **운동 완료 시**: 획득 포인트 애니메이션 (+100P 팝업)
- **스트릭 보너스**: 별도 축하 토스트 메시지

### 4.2 아이템샵 화면

- 카드 그리드 형태로 아이템 목록 표시
- 각 카드: 아이템 이름, 가격, 구매 버튼
- 포인트 부족 시 버튼 비활성화 + "OOO P 부족" 표시
- 상단에 현재 보유 포인트 표시

### 4.3 내 인벤토리 화면

- 보유 쿠폰 리스트 (미사용만 기본 표시)
- "사용하기" 버튼 → 확인 다이얼로그 → 파트너 알림
- 필터: 전체 / 미사용 / 사용완료
- 환불 버튼 (미사용 쿠폰만)

### 4.4 포인트 히스토리

- 시간순 내역 리스트
- 각 항목: 날짜, 사유, +/- 포인트, 잔액
