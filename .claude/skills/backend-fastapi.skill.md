# 백엔드 FastAPI 스킬

FastAPI, PostgreSQL, AI 서비스를 사용한 백엔드 개발을 관리합니다.

## 작업 흐름
1. Pydantic으로 API 스키마를 정의합니다.
2. 부부 피트니스 데이터를 위한 데이터베이스 모델을 생성합니다.
3. 개인화 추천을 위해 OpenAI를 통합합니다.
4. 엔드포인트 및 모델용 Python 코드를 출력합니다.

## 자산
- FastAPI 라우터 템플릿.
- PostgreSQL용 SQLAlchemy 모델.
- OpenAI API 통합 예시.
