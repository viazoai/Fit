# 프론트엔드 React 스킬

TypeScript, Vite, PWA 설정을 포함한 React 컴포넌트 개발을 지원합니다.

## 작업 흐름
1. 디자인 스펙이나 요구 사항을 수신합니다.
2. shadcn/ui를 이용해 React 컴포넌트를 생성합니다.
3. 모바일 우선 반응형 클래스 적용합니다.
4. 한국어 텍스트와 부부 특화 UX를 포함합니다.
5. 임포트가 포함된 완전한 컴포넌트 코드를 출력합니다.

## 자산
- React hooks, TypeScript 타입.
- shadcn/ui 컴포넌트(버튼, 폼 등).
- PWA 매니페스트 및 서비스 워커 템플릿.
