# 디자이너 Shadcn 스킬

Fit 프로젝트에서 shadcn/ui 컴포넌트를 사용한 UI/UX 설계를 전문적으로 지원합니다.

## 작업 흐름
1. 부부 중심 피트니스 앱 요구 사항을 분석합니다.
2. shadcn/ui 토큰(색상, 간격, 타이포그래피)을 적용합니다.
3. 모바일 우선 반응형 디자인을 확인합니다.
4. Figma 스타일 컴포넌트 스펙 또는 shadcn 코드 스니펫을 출력합니다.

## 자산
- shadcn/ui 컴포넌트 라이브러리.
- Polar 프로젝트 디자인 참조.
- 모바일 브레이크포인트: 기본 375px, md 768px.
