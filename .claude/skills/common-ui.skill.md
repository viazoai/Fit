# 공통 UI 스킬

Fit 프로젝트 전반에서 공통 UI 지침을 공유하여 디자인 및 프론트엔드 작업의 일관성을 확보합니다.

## 가이드라인
- **모바일 우선**: 먼저 375px 크기로 설계하고, `md:` 768px 브레이크포인트를 적용합니다.
- **shadcn/ui 규칙**: `asChild` prop 사용 금지, 링크에는 `buttonVariants()`를 사용합니다.
- **모바일 감지**: `window.innerWidth` 대신 `useIsMobile()` 훅을 사용합니다.
- **언어**: 모든 UI 텍스트는 부부 사용자를 위해 한국어로 작성합니다.
- **목표**: 개인정보 보호 중심, 개인화된 부부 피트니스 경험.

## 자산
- 참조: CLAUDE.md 디자인 시스템 섹션.
- 예시: 버튼 변형, 모바일 UX 우선 폼 레이아웃.
